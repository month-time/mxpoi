关于border属性

2017-08-04

今天被dalao问到一个问题 当一个盒子元素的宽高内边距都是0的时候，border:1px solid #000会有什么效果 当时我是一脸懵逼的。 回去仔细查询了一下资料发现border可以直接绘制，并不需要盒子模型其他属性的依赖……果然我还是有很多不足的地方….. 所以border:1px solid #000 依旧会绘制出一个1px的像素点 回答没有体积所以不会显示的我宛如ZZ！ border也可以用来绘制三角形图案 通过分别设置四个边框的宽度和透明度即可达到效果 我理解的原理大概是 每条边其实都是相连的 当一条边的长度为0的时候，其相邻的两条边就会连接到其神展的原点，这样就自动形成了一个三角形；

/\* 正三角*/
.abc1{
    width:0px;
    height:0px;
    padding: 0px;
    border: 1px solid #000;
    border-width: 0 50px 50px 50px;
    border-color:#333 transparent ;
}
/* 向右的三角*/
.abc2{
    width:0px;
    height:0px;
    padding: 0px;
    border: 1px solid #000;
    border-width:50px 0 50px 50px;
    border-color:transparent #333;
}
/* 倒三角*/
.abc3{
    width:5px; height:5px; padding: 0px; 
    background-color: aqua;
    border: 1px solid #000;
    border-width:50px 50px 0 50px;
    border-color: #333  transparent;
}
/* 向左的三角*/
.abc4{
    width:0px; height:0px; padding: 0px;
    border: 5px solid #000;
    border-width:50px 50px 50px 0;
    border-color: transparent #333;
}
</style>
